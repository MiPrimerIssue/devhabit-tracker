# ✅ Task 001 - Habit Card Component

## Nivel

Beginner

---

## 🎯 Objetivo

Crear una tarjeta reutilizable para mostrar un hábito con su estado y frecuencia.

---

## 🧠 Contexto

Esta tarea forma parte de la primera versión del proyecto y está pensada para construir una base visual clara, reutilizable y fácil de mantener.

---

## ✅ Requisitos

- La tarjeta debe mostrar nombre del hábito, frecuencia y estado
- Debe ser reutilizable
- Debe ser responsive
- Debe tener jerarquía visual clara
- Debe respetar el estilo general del proyecto

---

## 📁 Archivos sugeridos

```text
src/components/habit-card.tsx
src/app/page.tsx
```

---

## ✔️ Criterios de aceptación

- La tarjeta renderiza correctamente
- La información es clara
- Se adapta a móvil y escritorio
- Es reutilizable
- El cambio está centrado solo en esta tarea

---

## ✨ Extra opcional

- Añadir icono según estado
