# 📊 Task 002 - Progress Summary

## Nivel

Beginner

---

## 🎯 Objetivo

Crear un bloque visual de resumen con progreso diario o semanal.

---

## 🧠 Contexto

Esta tarea forma parte de la primera versión del proyecto y está pensada para construir una base visual clara, reutilizable y fácil de mantener.

---

## ✅ Requisitos

- Crear un componente de resumen visual
- Debe mostrar información simple de progreso
- Debe ser claro y reutilizable
- Debe adaptarse bien a distintos tamaños de pantalla
- Debe mantener coherencia visual

---

## 📁 Archivos sugeridos

```text
src/components/progress-summary.tsx
src/app/page.tsx
```

---

## ✔️ Criterios de aceptación

- El resumen se muestra correctamente
- La información se entiende de un vistazo
- Se adapta bien a móvil y escritorio
- Es reutilizable
- El cambio está centrado solo en esta tarea

---

## ✨ Extra opcional

- Añadir estados visuales positivos cuando hay progreso
