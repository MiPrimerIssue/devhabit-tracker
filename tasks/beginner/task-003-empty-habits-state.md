# 🌱 Task 003 - Empty Habits State

## Nivel

Beginner

---

## 🎯 Objetivo

Crear un estado vacío para cuando el usuario todavía no haya creado hábitos.

---

## 🧠 Contexto

Esta tarea forma parte de la primera versión del proyecto y está pensada para construir una base visual clara, reutilizable y fácil de mantener.

---

## ✅ Requisitos

- Crear un componente de estado vacío
- Debe incluir mensaje claro y motivador
- Puede incluir icono o ilustración sencilla
- Debe ser reutilizable
- Debe seguir el estilo del proyecto

---

## 📁 Archivos sugeridos

```text
src/components/empty-habits-state.tsx
src/app/page.tsx
```

---

## ✔️ Criterios de aceptación

- El mensaje es claro y útil
- El componente se ve limpio
- Es reutilizable
- Se adapta bien a diferentes tamaños de pantalla
- El cambio está centrado solo en esta tarea

---

## ✨ Extra opcional

- Añadir botón de CTA para crear el primer hábito
